Данное приложение это архив книг в который можно добавлять, удалять и искать добавленные книги в списке.

Команда разработчиков:
1. Александра Ушакова
2. Мина Степан
3. Александр Пожарский 

Инструкция по использованию:
1. Скачиваете 3 файла
2. Добавляете их в 1 папку 
3. Открываете пайчарм
4. Запускаете файл main.py
5. Нажимаете на кнопку run
